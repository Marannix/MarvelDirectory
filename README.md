# MarvelDirectory
 
**What is this?**

Displaying marvel characters and comics using Marvel API while using a collection of Android Architecture Components. 

**Android Architecture Components and libraries used**

```
- Room
- Navigation
- Retrofit
- Paging
- RxJava 
- Picasso
- Lottie
```
